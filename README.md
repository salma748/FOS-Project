# FOS Project

## Course
Operating Systems

## Overview
This project is an educational operating system developed as part of the
Operating Systems course. It focuses on implementing core OS concepts including memory
management, process scheduling, and synchronization.

## Implemented Modules
- Dynamic Memory Allocator
- Kernel Heap
- Page Fault Handling (Placement & Replacement)
- User Heap
- Shared Memory
- CPU Scheduling 
- Kernel Protection & Synchronization

## Tools & Technologies
- C Language
- GNU Make
- Bochs Emulator
